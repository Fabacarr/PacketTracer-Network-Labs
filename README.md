# PacketTracer-Network-Labs
Hands-on network design and configuration labs using Cisco Packet Tracer. 

ipv6_network: simple enterprise network using only ipv6 addresses for the hosts and routers. set up static routes to enable communication between available networks.

simple enterprose net: a large scale enterprise network in active development, still working on it and wannna make few upgrades. configured vlans to imptove network performance and setup layer 3 switches to handle inter vlan routing, layer 3 switches acting as dhcp servers to lease IPs to clients. deployed servers and made sure communication is possible. network still under active development, improvements will be made soon.

internetwork: configured SVI's with ip addressses on all switches to enable remote access, assigned end hosts with ipv4 addresses, secured all network devices with login credentails to prevent un authorized access. and the network is segmented into subnets to improve performance. static routes already configured to enable communication between networks.
